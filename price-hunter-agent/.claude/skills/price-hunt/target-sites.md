# 調査対象サイト一覧

price-hunt が価格収集フェーズ（SKILL.md ステップ2）で横断調査するサイトのリスト。API連携ではなくWebSearch/WebFetchで都度調べる方式のため、**登録できるサイト数に技術的な上限はない**。ここに追記すれば恒久的に対象へ加わる。多すぎて毎回全部を深掘りすると時間がかかるので、商品カテゴリに応じて関連性の低いものは軽く流してよい（例: 食品を探すのにイオシスは見なくてよい）。

## カード連携モール（ユーザー保有カードに対応・必須調査対象）
`user-profile.md` の保有カードに対応する自社/提携モール。カードのポイント還元とモール側のポイント還元が二重に乗ることが多いため、大手ECモールと同格の必須調査対象として扱う。

- au PAYカード → au PAY マーケット
- ビューカード（JR東日本） → JR Eモール（JRE MALL）
- イオンカード → イオンスタイルオンライン
- エポスカード → マルイウェブチャネル（マルコとマルオの7日間などのイベントに注意）
- 三井住友カード → Vポイント提携先（対象がECモールというよりコンビニ等の実店舗中心のため、通販商品の場合は対象外になりやすい点に注意）
- PayPayカード → Yahoo!ショッピング/PayPayモール（大手ECモールの節ですでにカバー）
- 楽天カード → 楽天市場（大手ECモールの節ですでにカバー）

新しいカードが `user-profile.md` に追加されたら、そのカードに対応する自社/提携モールがないかを確認し、このセクションに追記する。
