# price-hunter-agent

欲しい商品を型番まで突き詰め、複数のECサイトを横断比較して「実質最安値」を根拠付きで教えてくれるAIエージェント。第1弾はClaude Codeのスキルとして実装している。

## 使い方

1. このディレクトリ（`price-hunter-agent/`）で Claude Code を開く
2. `/price-hunt <商品名や欲しいものの説明>` を実行
   - 例: `/price-hunt ソニーのノイズキャンセリングイヤホン WF-1000XM5`
   - 型番が未確定でも可。対話で条件を詰めてから比較に進む
3. Amazon・楽天市場・Yahoo!ショッピング・価格.com・メーカー直販などを横断調査した結果が、実質価格（送料・ポイント還元込み）の比較表と結論として返ってくる

スキルの中身は [`.claude/skills/price-hunt/SKILL.md`](.claude/skills/price-hunt/SKILL.md) にある。

## 設計方針

`SKILL.md` に「商品特定 → 価格収集 → 実質価格の正規化 → 検証（詐欺サイト・型番違いの排除）→ 根拠付き提示」の5ステップを明文化している。1〜2サイトだけ見て終わらせず、複数カテゴリのサイトを横断することを必須にしている。
